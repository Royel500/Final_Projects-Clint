# **Delivery Management System**  
Live Link = [https://final_projects-rs-.web.app]
## **Project Description**  

The **Delivery Management System** is a full-stack web application designed to streamline parcel delivery operations for businesses, riders, and customers. It provides real-time tracking, automated rider assignment, payment processing, and analytics for efficient logistics management.  

### **Key Features**  

#### **1. Admin Dashboard**  
- **Real-time Analytics**: Track total riders, clients, parcels, and payments.  
- **User Management**: Manage roles (Admin, Rider, Client).  
- **Parcel Assignment**: Automatically assign riders based on location.  
- **Payment Tracking**: Monitor transactions and revenue.  
- **Activity Logs**: View recent system activities.  

#### **2. Rider Features**  
- **Delivery Assignments**: View assigned parcels and update status.  
- **Route Optimization**: Get optimized delivery paths.  
- **Earnings Dashboard**: Track completed deliveries and payments.  

#### **3. Client Features**  
- **Parcel Booking**: Create and track delivery requests.  
- **Payment Integration**: Secure checkout via Stripe.  
- **Delivery History**: View past and current deliveries.  
- **Rating System**: Rate riders after delivery.  

### **Technology Stack**  

#### **Frontend**  
- **React.js** (UI Framework)  
- **Tailwind CSS** (Styling)  
- **React Router** (Navigation)  
- **Axios** (API Calls)

#### **Backend**  
- **Node.js & Express.js** (Server)

- **Firebase Auth** (User Authentication)  
- **Stripe API** (Payments)  

#### **DevOps & Hosting**

- **Vercel/Netlify** (Frontend Hosting)
